# ops-training-template

未経験から Web運用保守に応募するための、擬似実務アウトプット（Incident / Runbook / Postmortem）を蓄積するテンプレートです。

このリポジトリは「テンプレートとして複製」して使います。

---

## Quick Start（5分）

### 1. テンプレートから自分のポートフォリオRepoを作る
1) 右上の **Use this template** をクリック  
2) Repository name を入力（例：`ops-training-portfolio`）  
3) 作成（Public推奨）

### 2. 最初のIncident（Lite）を1本書く
1) `docs/templates/incident_lite_template.md` をコピー  
2) 自分のIncidentとして保存：`docs/incident/incident_001.md`

### 3. 公開して“証拠”にする
- Public にするとURLで共有できます（応募時の提示に使えます）
- まずはPrivateで練習して、慣れたらPublicでもOKです

---

## Folder Structure
- `docs/incident/` : Incident Report を置く
- `docs/runbook/` : Runbook を置く
- `docs/postmortem/` : Postmortem を置く
- `docs/templates/` : テンプレート
- `docs/examples/` : 完成例

---

## Public公開時の注意（重要）
- 個人情報（氏名・メール・電話）を書かない
- 実在企業名・実案件の機密情報を書かない
- 架空のサービス名で統一する
- 事実（Timeline）と推測（Hypotheses）を混ぜない
